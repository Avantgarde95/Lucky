# Lucky

I'll predict your luck today.

### About
- Simple program I wrote a few years ago.
- Written in python.

### Requirements
- Python 2 or 3 (Both versions work!)

### Language
- The sentences are in Korean.
- If you want to change the sentences, modify `STRING_MAP` variable in the code.
